The Switch to GitHub Pages and Jekyll

17 Aug 2013

I wasn’t exactly happy with tumblr, so I went looking for alternatives to nothing really. Then I came across this blog post and noticed that it was powered by GitHub pages and Jekyll. I knew about GitHub pages but sort of thought they were relegated to repo descriptions and creating your own themes.

So I forked the @sirupsen blog and renamed it. Then I was off to the races. You can learn more about Jekyll, which has been a dream to work with. Most of my pages and all of my blog posts are written using Kramdown flavored Markdown. Leaving me with only a little HTML and very slight bit of CSS work to do.

You can clone or fork my blog on GitHub.
